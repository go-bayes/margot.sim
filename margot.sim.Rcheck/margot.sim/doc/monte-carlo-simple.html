Monte Carlo Evaluation - Simple Example

library(margot.sim)
set.seed(2024)

Introduction

This vignette provides a simple example of using the Monte Carlo framework in margot.sim to evaluate estimator performance.

Basic Monte Carlo Simulation

# Define a simple regression estimator
simple_regression <- function(data) {
  # Check if we have the required variables
  if (!all(c("t3_y", "t2_a", "b1", "b2", "b3") %in% names(data))) {
    return(list(estimate = NA, se = NA, converged = FALSE))
  }
  
  # Fit model
  fit <- lm(t3_y ~ t2_a + b1 + b2 + b3, data = data)
  
  # Extract coefficient
  if ("t2_a" %in% rownames(coef(summary(fit)))) {
    coef_summary <- summary(fit)$coefficients["t2_a", ]
    return(list(
      estimate = coef_summary["Estimate"],
      se = coef_summary["Std. Error"],
      converged = TRUE
    ))
  } else {
    return(list(estimate = NA, se = NA, converged = FALSE))
  }
}

# Run Monte Carlo simulation
mc_results <- margot_monte_carlo(
  n_reps = 100,
  n_per_rep = 500,
  dgp_params = list(
    waves = 2,
    treatments = "a",
    interventions = list(natural = function(data, time, trt) data[[trt]]),
    params = list(a_y_coef = 0.5)
  ),
  estimator_fn = simple_regression,
  seed = 123
)

Understanding the Results

The Monte Carlo results include: - Summary statistics: Mean estimate, bias, variance, RMSE across replications - Performance metrics: Coverage of confidence intervals, convergence rate - Individual results: Estimates from each replication (if needed for diagnostics)

Adding Shadows

You can evaluate how observational distortions affect your estimator:

# Create measurement error shadow
me_shadow <- create_shadow(
  type = "measurement_error",
  params = list(
    error_type = "classical",
    variables = "t2_a",
    sigma = 0.3
  )
)

# Run with shadow
mc_with_error <- margot_monte_carlo(
  n_reps = 100,
  n_per_rep = 500,
  dgp_params = list(
    waves = 2,
    treatments = "a",
    interventions = list(natural = function(data, time, trt) data[[trt]]),
    params = list(a_y_coef = 0.5)
  ),
  shadows = list(me_shadow),
  estimator_fn = simple_regression,
  seed = 456
)

Summary

The Monte Carlo framework helps you: 1. Evaluate estimator performance under known truth 2. Compare methods under different conditions 3. Understand the impact of observational challenges 4. Make informed choices about statistical methods

For more advanced examples, see the full Monte Carlo vignette.
